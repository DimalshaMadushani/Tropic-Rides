# ST Scooter Rentals (Next.js + Tailwind)

Redesigned version of **ST Scooter Rentals** built with **Next.js (App Router)**, **TypeScript**, and **Tailwind CSS**.

## Pages
- `/` Home (services overview, why choose us, testimonials, FAQ)
- `/scooter-rent`
- `/car-rent`
- `/tour`
- `/about`
- `/contact-us`

## Run locally
```bash
npm install
npm run dev
```

## Notes

  - `app/contact-us/page.tsx`
